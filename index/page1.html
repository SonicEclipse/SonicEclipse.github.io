Sonic Eclipse is a fan-made Sonic the Hedgehog game currently under development.
It's crafted by a dedicated team of Sonic enthusiasts aiming to blend the essence of classic Sonic games with fresh, modern gameplay mechanics.

Please note that this game is not affiliated with SEGA and is purely a fan project. Note that the game is still in development, and some features may not be fully implemented.

You can find the latest version of the game on the Downloads page.
For updates and announcements, please check our Discord server.
